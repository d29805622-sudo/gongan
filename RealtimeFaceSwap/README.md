# RealtimeFaceSwap

实时AI视频处理软件。

## 功能

- 摄像头实时采集
- 实时视频传输
- AI人脸处理
- GPU加速
- Flutter客户端

## 技术

Backend:

- Python
- FastAPI
- OpenCV
- WebSocket

Frontend:

- Flutter

AI:

- InsightFace
- ONNX Runtime

## 启动后端

进入：

```
backend
```

安装：

```
pip install -r requirements.txt
```

运行：

```
python app.py
```
